## Description
Rad-Bot is a Discord bot that offers many cool features for our community discord server.

## Requirements
* Python 3.8.2+
* discord.py 1.3.4+

## Commands
In the following list the default prefix `!` is used, but it can changed in the config.ini file.
* `!help`
* `!version`
* `!source code`
* `!whats new?`
* `!rad meme`
* `!how many rad memes?`
* `!sound like radovan`
* `!sound like rado`
* `!sound like rad`
* `!rhyme with radovan`
* `!rhyme with rado`
* `!rhyme with rad`
